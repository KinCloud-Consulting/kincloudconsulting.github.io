Privacy Policy

Effective date: 2025-07-19

1. Purpose

QBO Invoice Line Extraction is an internal tool used by KinCloud Consulting Inc. to read accounting data from our own QuickBooks Online tenant for reporting and analysis.

2. Data We Access

  • Accounting entities exposed by the QuickBooks Online API (e.g., invoices, customers, chart of accounts).
  • OAuth 2.0 access and refresh tokens required to authenticate.

3. Data Storage & Retention

  • Access tokens are kept only in local memory / encrypted local storage while the script runs.
  • No QuickBooks data is stored or persisted after the export completes.

4. Data Sharing

We do not share, sell, or disclose any QuickBooks data—or user‑identifiable information—to third parties.

5. Security

  • All API traffic is encrypted via HTTPS/TLS.
  • The script runs on an isolated machine behind a firewall and VPN.

6. Your Rights

Because the tool is used solely by Company staff on Company data, no customer personal data is processed. If you have questions, contact support.

7. Changes

We may update this policy and will post any changes on this page with a new effective date.
